# symmetrical-memory
Node.js REST API (Todo app) with Express. Basic backend structure with routes, controllers and in-memory data storage. Built for learning REST API design and backend fundamentals.

# Node.js Todo REST API

Simple backend project built with Node.js and Express.

## Stack
- Node.js
- Express

## Features
- Create todo
- Get all todos
- Toggle todo status

## Endpoints

GET /api/todos
POST /api/todos
PATCH /api/todos/:id

## Run

npm install
node src/server.js
